📡 Telecom Engineering Projects Portfolio

📌 Overview

This repository contains a collection of projects focused on telecommunications, signal processing, and network systems.

Each project explores a different layer of communication systems — from signal-level processing to network-level performance analysis.

Together, they demonstrate practical understanding of how modern communication systems are designed, analyzed, and optimized.



 🧠 Projects Summary

 📶 1. Telecom System Simulation

Focus: End-to-end digital communication system

This project simulates a complete communication pipeline:

Message → Encoding → Modulation → Noisy Channel → Demodulation → Decoding

 🔍 What it solves:

* How noise affects transmitted signals
* How errors occur during communication
* How error correction improves reliability

 ⚙️ Key Concepts:

* BPSK & QPSK modulation
* AWGN (noise modeling)
* Hamming error correction
* Bit Error Rate (BER) analysis

 🎯 Outcome:

Demonstrates the trade-off between efficiency and reliability in communication systems.



 📊 2. Signal Processing System

Focus: Signal filtering and noise analysis

This project studies how different filtering techniques improve signal quality under noisy conditions.

 🔍 What it solves:

* How noise distorts signals
* How filtering techniques recover useful information
* Which filters perform best under different conditions

 ⚙️ Key Concepts:

* Gaussian noise modeling
* Moving average filtering
* Butterworth filtering
* Frequency domain analysis (FFT)
* Signal-to-Noise Ratio (SNR)
* Mean Squared Error (MSE)

 🎯 Outcome:

Shows how signal quality can be restored and evaluated using engineering techniques.



🌐 3. Network Performance Simulation

Focus: Network behavior and performance evaluation

This project simulates how data flows through a network under different conditions.

🔍 What it solves:

* How congestion affects data transmission
* How bandwidth and queues impact performance
* How networks behave under load

⚙️ Key Concepts:

* Packet transmission
* Queue modeling
* Bandwidth limitations
* Delay, throughput, and packet loss

 🎯 Outcome:

Provides insight into how real-world networks manage traffic and performance.
